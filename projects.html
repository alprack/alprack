COVID-19 Relationship to Hospital Locations

Hospital Density

A map depicting the density of hospitals in the U.S.

This project is to collected info about how many cases of COVID there were in an area in comparison to the amount of hospitals in those areas in order to allow that information to be used by others for multiple purposes. For example, seeing if cases were more common with more hospitals around or to maybe compare this information to areas where deaths from COVID occur to see if there is a correlation with the available hospitals.
